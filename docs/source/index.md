>泡馅水饺是北京石林先生一九九五年十月向中国专利局提交书面申请和有关技术资料，通过五年的努力于2000年5月正式荣获中国专利号：95103873·7 泡馅水饺从制作到成品，工艺流程复杂，用料考究，运用十几种天然中草药材作调料，根据人体营养的需要，四季温度的不同，进行合理搭配，混合加工粉碎成复合型调料后酿制成大全料酒，在各种肉类没加工粉碎之前先把肉块在料酒里面进行淡泡，入味后再加工粉碎。这样对馅类可起到：去腥、报香、开胃、提神的作用。泡馅水饺达到了：百家之长，为我所用，精艺求精，永无止境。有请赞日：泡馅水饺精肉馅，吃到嘴里一肉蛋，大葱生姜细挑选，三十六种药材面，香而不腻回味长，中外宾客齐赞扬。
