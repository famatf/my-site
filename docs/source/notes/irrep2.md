# 表示的完备性
## 总结
* $\sum_i m_i^2 = g$ 说明在表示空间内有 $g$ 个线性无关的矢量：
  * 矩阵元是完备的： $F(R) = \sum_n C_n^jD^j(R)_{\mu\nu}$
  * 表示矢量是完备的： $U_{j\mu\nu}=\sum D^j(R)_{\mu\nu} \qquad X = \sum_{j\mu\nu}C^j_{\mu\nu}U_{j\mu\nu}$
  * 类矢量是完备的（特征标完备）：
    * 类函数 $F(R)=\sum_j C_j \chi^j(R) \qquad O=\sum_jC_jV_j$
  * $\sum_j 1=g_c$
*  不可约表示 $\Longleftrightarrow \sum_R \chi^*(R)\chi(R)=g$
*  表示等价 $\Longleftrightarrow$ 特征标相同
